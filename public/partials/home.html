Jacob Waterman's MEAN Task Manager

MP3 from UIUC's CS 498Rk - The Art and Science of Web Development

Feel free to explore. You can create users and assign them tasks. Then mark those tasks down as completed once they are finished. Enjoy!


The data all comes from my own backend, the API can be seen at http://jwaterman-todo.herokuapp.com/api and the documentation for the api can be found on the Github page.
